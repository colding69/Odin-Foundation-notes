What is the difference between Git and GitHub?
Git is a localised program
GitHub is a remote online platform



Why is Git useful for a team of developers as a distributed VCS?
Prevent different individuals to add same changes on the same file.
Merges edits from different individuals done to the same project in a compatible form.


How does a fork differ from a branch?
A branch is still a version of the same repository belonging to the same account.
A fork is a new repository created from a template, belonging to a different account.
