SCENARIO: Each button is set to 200px wide. However, the width of dotted black container is only 600px, meaning all 4 buttons shrink to fit container. How to get the buttons to overflow instead of shrink. i.e. move to newline if 1 line of container is not long enough?

multiline container

SCENARIO: You want to use flex-flow which specifies [flex-direction] [flex-wrap] to create a vertical block with wrap to overflow if the height of container is not long enough to house all buttons.
