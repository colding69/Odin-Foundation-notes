Effect of flex-direction

The properties for x and y axis alignment are switched around when you change flex-direction.

Default flex-direction is row

  • justify-content aligns along x axis
  • align-items aligns along y axis

What happens when flex-direction is set to column?

When flex-direction is set to column. THINGS SWAP!

  • justify-content aligns along y axis
  • align items aligns along x axis
